# **App Name**: LearnAI

## Core Features:

- User Registration: Allow new users to register with email, password, name, phone, address, role, state, and face capture.
- Role-Based Login: Enable login for students, teachers, and admin with role-based dashboards.
- Student Dashboard: Provide modules for courses, live classes, recorded lectures, AI chatbot, announcements, face attendance, assignments, chat history, and profile management.
- AI Teacher: Automatically conduct virtual classes, explain topics, and interact with students using AI.
- AI Recorded Lectures: Offer access to recorded AI-conducted sessions for anytime learning.
- AI Chatbot: Provide a 24/7 AI assistant for doubt-solving and instant support. This tool should use its judgement to decide when it needs to bring in existing resources for use in answering a student's question.
- Admin Panel: Control system access, manage users, roles, and monitor system activities.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to convey trust, reliability, and knowledge, reflecting the educational context.
- Background color: Light blue (#D4E0F7), a very desaturated version of the primary color to provide a calm learning environment.
- Accent color: Violet (#8E44AD) to highlight important elements and calls to action, while complementing the primary color.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and tech-forward feel.
- Body font: 'Inter' (sans-serif), for a clean and readable design
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent and clear icons for navigation and features, following a minimalist style.
- Implement subtle animations for user feedback and transitions to enhance the learning experience.
